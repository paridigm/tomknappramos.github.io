Game of Coral Life




Each player chooses a coral species from branching coral, plating coral, massive coral, or encrusting coral.

Each game board tile causes a random event that can either promote coral growth or cause coral bleaching.Players compete by hitting numbers on a slider in attempt to land on good events generated by bright blue game board tiles.

My part of the project was the game engine design including all board game mechanics, physics, in-game play, and the coral event card system.

CONTROLS (Xbox):
A,S,W,D Keys (Left Joystick) - Character Selection
SPACE (A) - Select
ENTER (START) - Start Game
