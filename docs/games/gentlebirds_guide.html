A Gentlebird's Guide to Dating and Mating


http://globalgamejam.org/2016/games/gentlebirds-guide-dating-and-mating

This game takes the player through several mini-games tied together as steps in "A Gentlebird's Guide To Dating and Mating", a definitive guidebook to getting an egg laid.

It was made at the Honolulu Global Game Jam 2015 using the Unity Game Engine. My part of the project was the design of the dance floor scene and the programming behind the dancing engine.

CONTROLS:
Arrows - Movement and Dancing
